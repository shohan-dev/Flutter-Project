### Implicit Animations 🌀

Simple animations with minimal code, ideal for known start and end values.

**Examples:**
- `AnimatedContainer`
- `AnimatedOpacity`
- `AnimatedAlign`



---

### Explicit Animations 🎨

Full control over animations using `AnimationController` and `Tween`.

**Examples:**
- `AnimatedBuilder`
- `Transition`
